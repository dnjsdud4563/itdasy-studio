# 인프라 비용 · 사용량 · 한도 정리

> 각 서비스의 무료 한도, 실제 제약, 스케일 시 전환 전략을 기록합니다.

---

## 서비스별 현황

### 1. Supabase (DB + Auth + Storage)
- **역할**: 데이터베이스, 사용자 인증, Edge Functions
- **플랜**: Free
- **한도**: DB 500MB, Auth 5만 요청/월, Edge Function 50만 호출/월
- **초과 시**: Pro $25/월
- **전환 시점**: 유저 ~1,000명 (DB 300MB+ 도달 시)

### 2. Cloudflare R2 (이미지 스토리지)
- **역할**: AI 생성 이미지, 사용자 업로드 이미지 저장
- **플랜**: Free (카드 등록 필수, 청구는 초과분만)
- **한도**: 저장 10GB/월, 쓰기 100만 회/월, 읽기 1000만 회/월, **다운로드 트래픽 무제한 무료**
- **초과 시**: 저장 GB당 $0.015, 쓰기 100만당 $4.50
- **전환 시점**: 유저 ~5,000명 (10GB 도달 시 → 요금 발생하지만 매우 저렴)

### 3. Pollinations (AI 이미지 생성)
- **역할**: 텍스트 프롬프트 → AI 이미지 생성
- **플랜**: 완전 무료, API 키 불필요
- **한도**: 공식 제한 없음 (커뮤니티 서비스)
- **제약**: 속도 느림 (5~15초), 품질 Ideogram 대비 낮음, SLA 없음
- **전환 시점**: 이미지 품질/속도 불만 시 → Ideogram ($20 선불 충전)

### 4. HuggingFace Inference API (RMBG-2.0 배경 제거)
- **역할**: 이미지 누끼(배경 제거), HuggingFace 서버의 GPU에서 실행됨
- **왜 무료?**: AI 모델 홍보 + 개발자 생태계 확보 전략 (AWS 프리티어와 같은 논리)
- **실제 제약**:
  - 분당 ~30회 Rate limit
  - 모델 안 쓰면 잠듦 → 첫 요청 10~30초 Cold start
  - 동시 처리 1~2건 (큐 대기)
  - SLA 없음 (느려도, 터져도 보장 없음)
- **유저 100명 기준**: 일 300회 = 분당 0.2회 → **한도 내 여유**
- **전환 시점**: 유저 1,000명+ (분당 30회 제한 도달) → HuggingFace Pro ($9/월, 전용 GPU) 또는 자체 rembg 서버 (2GB VPS $7~20/월)

### 5. Fastify (API 서버 프레임워크)
- **역할**: 원영님 앱의 모든 요청을 받아서 처리하는 API 서버. Node.js 위에서 동작.
- **비용**: ₩0 (오픈소스)
- **호스팅 필요**: 아직 미배포. Render 무료(512MB, 750시간/월) 또는 Fly.io 사용 예정.

### 6. RevenueCat (인앱결제 관리) — 미연동
- **역할**: Apple/Google IAP 영수증 검증, 구독 상태 관리
- **플랜**: Free ($2.5M 매출까지)
- **연동 시점**: 앱마켓 등록 후

### 7. PortOne (국내 웹 결제) — 미연동
- **역할**: 한국 PG사 연동 (카카오페이, 토스 등)
- **플랜**: 테스트 무료
- **연동 시점**: 웹 결제 필요 시

## 데이터 흐름 (어디서 뭘 처리하는지)

```
원영 앱 (React Native)
  │
  ▼
Fastify API 서버 (우리 서버, Node.js)
  │
  ├─→ Supabase        : 로그인, DB 읽기/쓰기, 사용자 인증
  ├─→ Cloudflare R2   : 이미지 저장/조회 (다운로드 무료)
  ├─→ Pollinations    : AI 이미지 생성 (외부 서버에서 실행)
  ├─→ HuggingFace     : 배경 제거 (외부 GPU 서버에서 실행)
  ├─→ Instagram API   : IG 피드 조회 (Meta 서버)
  └─→ RevenueCat      : 결제 검증 (미연동)
```

핵심: **무거운 AI 연산은 전부 외부 서버(HuggingFace, Pollinations)에서 실행**. 우리 서버는 요청을 전달하고 결과를 저장하는 **중개자** 역할만.
